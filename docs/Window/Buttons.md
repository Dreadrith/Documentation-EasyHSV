# Buttons & Others
![More](/img/window_more.jpg)

- **Generate HSV(A) Coloring:** Applies EasyHSV to the Avatar and assets based on the settings and targets.
	- Shows a tooltip on hover if the button is not active.
		![Button](/img/window_button.jpg)
- **Trash Button:** Can be clicked if EasyHSV was previously added. Shows a menu to choose which previous HSV setups to remove.
- **Dreadscripts Banner:** Takes you to https://www.dreadrith.com/ on click.
- **Made by Dreadrith:** ...Who?
- **Heart Icon:** On click, shows the coolest people ever.
